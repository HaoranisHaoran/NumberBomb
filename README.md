# 数字炸弹 💣

一个极致简洁的猜数字游戏，用 Swift 编写，仅 121KB。

## ✨ 功能
- 四个难度：简单(1-50)、中等(1-100)、困难(1-500)、自定义
- 各难度独立最佳记录
- 中英文切换（系统自动翻译）
- iOS/macOS 双平台
- 帮助系统
- 系统原生动画

## 📱 平台
- iOS 16.0+
- macOS 12.3+
